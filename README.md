# Cloud

Orchestrates Rails+Kubernetes Clouds.

## Install

Copy and paste this:
```
curl -s https://raw.githubusercontent.com/chi-rb/chi-cloud/master/bin/cloud | sudo sh -s install
```

## Usage

Print manual using:
```
cloud help
```
